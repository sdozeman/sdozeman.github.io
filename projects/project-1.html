Wee Wiggles

Wee Wiggles Inc. is a small private dayhome in Alberta, Canada. The client needed a website that was small but usable and still displayed important information for parents regardless of screen size or device.

This project was built on WordPress and uses a single page with Advance Custom Fields to define areas of content. Some features include:

  • WordPress
  • HTML5
  • CSS3
  • LESS
  • JavaScript
  • jQuery
  • PHP
  • Responsive Design
  • Sticky Navigation
  • ScrollTo Functionality
  • Keyboard Accessibility
  • Parallaxing Images
  • Image Gallery
  • Contact Form
  • Anti-Spam Protection
