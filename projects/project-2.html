Frank & Able

Family owned and operated Frank & Able tomatoes were in need of a web presence, something to help showcase their on-the-vine tomatoes, provide information about them as a company and family, and also to help connect with buyers, consumers, and the world.

Though this project is a single page web application currently, it was built on WordPress as a custom theme as there are plans for growth in the future.
Some features include:

  • WordPress
  • HTML5
  • CSS3
  • LESS
  • JavaScript
  • jQuery
  • PHP
  • Full Screen Background Video
  • Responsive Design
  • ScrollTo Functionality
  • Keyboard Accessibility
  • Contact Form
  • Anti-Spam Protection
