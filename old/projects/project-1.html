Oppy Transport

Oppy Transport is a professional transportation service provided by The Oppenheimer Group (Oppy), as industry experts they needed a web presence that was informative, functional, modern and usable anytime, anywhere.

This project was built as a custom theme on WordPress using Advanced Custom Fields. Some features include:

  • WordPress
  • HTML5
  • CSS3
  • LESS
  • JavaScript
  • jQuery
  • PHP
  • Responsive Design
  • Sticky Navigation
  • Keyboard Accessibility
  • Custom Blog/News Templates
  • Fade-In Content
  • Contact Form
  • Anti-Spam Protection
